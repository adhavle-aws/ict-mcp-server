Create Apex Action for Agentforce

Create an Apex class that Agentforce can invoke:

public class AWSArchitectAction {
    @InvocableMethod(label='Generate AWS Infrastructure' 
                     description='Generate CloudFormation template')
    public static List<Result> generateInfrastructure(List<Request> requests) {
        Request req = requests[0];
        
        // WebSocket connection details
        String wsUrl = 'wss://9h32jjkhg3.execute-api.us-east-1.amazonaws.com/prod';
        
        // Prepare MCP request
        Map<String, Object> payload = new Map<String, Object>{
            'id' => String.valueOf(DateTime.now().getTime()),
            'tool' => 'build_cfn_template',
            'arguments' => new Map<String, Object>{
                'prompt' => req.requirements,
                'format' => 'yaml'
            }
        };
        
        // Call WebSocket API (implement WebSocket client)
        String template = callWebSocketAPI(wsUrl, payload);
        
        Result result = new Result();
        result.template = template;
        result.success = true;
        return new List<Result>{ result };
    }
    
    public class Request {
        @InvocableVariable(required=true label='Infrastructure Requirements')
        public String requirements;
    }
    
    public class Result {
        @InvocableVariable(label='CloudFormation Template')
        public String template;
        
        @InvocableVariable(label='Success')
        public Boolean success;
    }
    
    private static String callWebSocketAPI(String wsUrl, Map<String, Object> payload) {
        // Implement WebSocket client logic here
        // Or use HTTP callout to a proxy Lambda
        return 'Template content';
    }
}

Register with Agentforce

  1. Go to Setup ‚Üí Agents ‚Üí Your Agent
  2. Click "Add Action"
  3. Select "Apex Action"
  4. Choose "AWSArchitectAction.generateInfrastructure"
  5. Configure input mapping
  6. Save and activate

Troubleshooting

Issue Solution
Remote Site not configured Add AWS_WebSocket_API to Remote Site Settings
WebSocket connection fails Verify URL: wss://9h32jjkhg3.execute-api.us-east-1.amazonaws.com/prod
Timeout errors Normal for complex templates (20-30 seconds)
Validation errors Auto-fix is enabled - template will be corrected automatically
